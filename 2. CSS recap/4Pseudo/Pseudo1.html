Opdracht 1

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)> Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Een paar voorbeelden van pseudo-elementen met hun uitleg zijn:

  1. ::before. Hiermee kun je content voor je element plaatsen.
  2. ::after. Hiermee kun je content na je element plaatsen.
  3. ::first-letter. Hiermee kun je de opmaak van de eerste letter van je element wijzigen.
  4. ::first-line. Hiermee kun je de opmaak van de eerst regel van je element wijzigen.
