Opdracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

Een CSS syntax heeft altijd 3 stijlregels, namelijk:
  • Selector
  • Property
  • Waarde

Als je de volgende CSS syntax hebt, dan is h3 de selector, want het gaat om de opmaak van de h3 in dit geval. De color en font-size zijn de properties, dat betekent wat je aan de h3 toe wilt wijzen, en de rood en 50px zijn de waardes die je aan de properties geeft.
