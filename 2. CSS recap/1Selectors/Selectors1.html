Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML laden. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoudt.

Je hebt drie verschillende manieren om de CSS in je HTML te laden, namelijk:
  • Inline CSS
  • Internal CSS
  • External CSS
