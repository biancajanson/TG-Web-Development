Leg in eigen woorden uit wat met cascade en inheritance wordt bedoeld. Maak voorbeelden om je antwoord uit te leggen.

Cascade

Bij cascade gaat het erom dat er conflicterende CSS properties zijn. Cascade geeft dan aan welke eigenschappen wel en welke niet aan een element toegewezen worden. Zie hier een voorbeeld.

In dit geval heb ik 2 keer een CSS propertie gegeven aan de p elementen, namelijk als eerste de kleur rood, en als tweede de kleur blauw. Door de cascade wordt altijd de laatste regel de uiteindelijke eigenschappen die mee gegeven worden. In dit geval wordt de eigenschap rood dus overgeschreven door de eigenschap blauw omdat deze lager in het CSS bestand staat.

Inheritance

Inheritance houdt in dat sommige CSS eigenschappen die zijn ingesteld op de bovenliggende elementen, de parent, wel overgenomen wordt door de child elementen, en andere niet. Zie hier een voorbeeld.

In dit geval heb ik het div element een eigenschap gegeven, namelijk de kleur rood. Omdat ik het laatste p element een id heb gegeven, wordt alleen bij dit element de eigenschap van de div niet overgenomen, maar behoudt dit element zijn eigen eigenschap.
