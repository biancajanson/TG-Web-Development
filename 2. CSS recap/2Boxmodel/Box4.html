Opdracht 4

Leg uit wat de volgende termen betekenen: content, padding, margin en border.

De betekenis van de termen:

  • Content: de inhoud van je element.
  • Padding: een vak dat om de inhoud van je element heen zit..
  • Margin: een onzichtbare rand die om je border heen zit, dat de afstand met andere elementen bepaalt.
  • Border: een rand om je element heen, dat de opvulling (de padding) van je element omringt.
