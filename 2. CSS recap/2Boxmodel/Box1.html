Wat zijn de eigenschappen van block boxes en inline boxes?

Een block box pakt de hele horizontale ruimte die beschikbaar is, ongeacht hoeveel het nodig heeft. Dit betekent daarmee ook automatisch dat een block-element dat erna komt automatisch op een nieuwe regel gezet wordt (tenzij je dit zelf verandert in de CSS properties).
Een inline box pakt daarentegen genoeg horizontale ruimte als er nodig is, in plaats van de hele breedte. Er kunnen dus meer inline elementen op een regel toegevoegd worden, tot deze regel vol is.
